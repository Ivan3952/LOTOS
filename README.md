# LOTOS Event Site Demo

Новый сайт на базе COLUMBINA, но в стиле Imperial Space Lotus.

Есть:
- главная LOTOS;
- intro;
- победители с пагинацией;
- все работы;
- режим галереи;
- случайная работа;
- музыка только по кнопке;
- комментарии к работам;
- lightbox с комментарием.

Формат data.js:
{
  id: 1,
  username: "@user",
  title: "Работа #1",
  image: "img/works/001_user.webp",
  postLink: "https://...",
  score: 9,
  reward: 1200,
  isWinner: true,
  place: 2,
  awardTitle: "Golden Lotus",
  comment: "Комментарий к работе."
}
