# LOTOS v2 — distinct style patch

Это не новый движок, а радикально другой CSS для демо LOTOS.

Что поменялось:
- вертикальная навигация слева вместо обычной верхней панели;
- главная как разворот артбука: текстовый дворцовый блок + большая обложка;
- карточки работ стали горизонтальными “свитками”;
- другие шрифты: Cormorant Garamond + Manrope;
- углы резаные, меньше неона, больше дворца/нефрита/алого лака;
- статистика стала одной длинной панелью;
- меню разделов стало дворцовыми табличками.

Как поставить:
1. В существующий LOTOS-демо загрузи `style-lotos-v2.css`.
2. В index.html замени:
   <link rel="stylesheet" href="style-lotos.css?v=1" />
   на:
   <link rel="stylesheet" href="style-lotos-v2.css?v=2" />

Если хочешь полностью отдельный вариант — можешь просто держать оба CSS и переключать ссылку.
